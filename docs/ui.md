# UI Components

Ringkasan komponen UI dan file terkait:

- [src/components/ui/button.tsx](src/components/ui/button.tsx): Pembungkus `Button` dengan variasi gaya menggunakan `class-variance-authority`.
- [src/components/ui/textarea.tsx](src/components/ui/textarea.tsx): Pembungkus `textarea` yang meneruskan `ref` dan styling konsisten.
- [src/components/ui/tooltip.tsx](src/components/ui/tooltip.tsx): Wrappers untuk primitive tooltip (`Tooltip`, `TooltipTrigger`, `TooltipContent`, `TooltipProvider`).
- [src/components/ui/sonner.tsx](src/components/ui/sonner.tsx): Toaster/notification wrapper yang mengonfigurasi tema dan ikon `sonner`.
- [src/components/ui/separator.tsx](src/components/ui/separator.tsx): Pembungkus `Separator` primitive.
- [src/components/ui/badge.tsx](src/components/ui/badge.tsx): Komponen `Badge` dengan varian gaya.
- [src/components/ui/avatar.tsx](src/components/ui/avatar.tsx): Grup avatar dan utilitas (`Avatar`, `AvatarImage`, `AvatarFallback`, `AvatarBadge`, `AvatarGroup`).

Setiap file berisi komentar header dan detail penggunaan singkat. Lihat file sumber untuk contoh penggunaan dan props.
